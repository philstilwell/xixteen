Production flow

How the site moves from code to visitors.

XiXteen is a static learning app with a small Cloudflare backend for scoring, profiles, and leaderboards. The quiz content is committed as data files; no paid AI API is needed at runtime.

01 Source

The GitHub repo stores the public site, the generated quiz corpus, daily quiz schedules, skill images, favicon, Pages Functions, D1 schema, and seed exporter.

  • index.html starts the public app.
  • data/question-bank.json holds 2,560 objective items.
  • data/daily-quizzes.json assigns one item per skill per day.

02 Deploys

GitHub Actions deploys a static backup to GitHub Pages. Cloudflare Pages is the production host because it can run the scoring Functions and bind to D1.

  • GitHub Actions runs the static Pages workflow on pushes to main.
  • Cloudflare Pages deploys from a clean commit with Wrangler.
  • xixteen.pages.dev remains the stable Cloudflare project URL.

03 Runtime

Visitors load static HTML, CSS, JavaScript, images, and JSON. When a public score is submitted, the browser calls Cloudflare Pages Functions under /api.

  • /api/score validates and records a completed daily quiz.
  • /api/leaderboard returns daily or weekly public rankings.
  • /api/profile returns per-skill stats for the local participant.

04 Database

Cloudflare D1 stores normalized skills, items, choices, daily quizzes, participants, score submissions, attempts, and skill stats.

  • The app scores against server-side correct answers.
  • Fast or duplicate submissions can be held out of public boards.
  • Skill stats update after accepted daily submissions.

05 DNS

Cloudflare is authoritative for xixteen.com. The apex and www hostnames point to the Cloudflare Pages project.

  • Namecheap nameservers: jack.ns.cloudflare.com and tia.ns.cloudflare.com.
  • xixteen.com is active with SSL enabled.
  • www.xixteen.com is also attached to the Pages project.

06 Privacy

The app avoids account creation. It keeps a local participant id in the browser and only asks for a display name when someone wants a public leaderboard score.

  • Leaderboard entries show display name, score, total, time, and submission time.
  • Profile stats are tied to the local participant id.
  • No runtime AI prompt or user answer is sent to an AI provider.

Operations

What to remember when changing the site.

Run npm run check before shipping corpus or UI changes. It regenerates, validates, and audits the 16-skill quiz set.

If the D1 seed format changes, run npm run db:seed, then apply the resulting SQL with Wrangler. Remote D1 rejects SQL transaction wrappers.

After a Cloudflare Pages deploy, test both the homepage and an API route. DNS caches may briefly show old GitHub records even after authoritative Cloudflare DNS is right.
